# Sistema Básico de Inventario

## Descripción
Este proyecto consiste en un programa en Python que permite registrar productos en un inventario simple.

El usuario debe ingresar:
- Nombre del producto
- Precio unitario
- Cantidad

El sistema valida los datos ingresados y calcula el costo total del inventario.

## Objetivos del proyecto
Aplicar fundamentos de programación como:

- Entrada de datos
- Variables
- Validación de datos
- Operaciones matemáticas
- Salida de información en consola

## Tecnologías utilizadas
- Python
- Git
- GitHub
